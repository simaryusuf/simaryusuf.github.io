C++ Hata Ayiklama (Debugging) Egzersizleri

Asagidaki kodlarin her birinde sadece bir tane mantik veya yazim hatasi bulunmaktadir. Hatayi bulduktan sonra dogru cevabi gormek icin "Cevabi Goster" butonuna tiklayin.

1. Soru (Karekok ve Veri Tipi)

#include <iostream>
#include <cmath>

using namespace std;

int main()
{
   int i;
   int x = 20;
   int y = 0;
   double z;

   for (i = 1; i < 4; i++)
   {
         y = x + i;
   }

   if (x == 20)
   {
       int z = sqrt(x); 
       cout << "Karekok: " << z << endl;
   }

   return 0;
}
Cevabi Goster

Hata: if blogunun icinde int z = sqrt(x); yazilmis. z zaten yukarida double olarak tanimlanmisti ve sqrt ondalikli bir deger dondurur.

Cozum: Bastaki int kelimesi silinmeli: z = sqrt(x);

2. Soru (Dongu Yonu)

#include <iostream>
#include <cmath>

using namespace std;

int main()
{
   int i;
   int x = 1;
   int y = 0;
   double z = 0.0;

   for (i = 1; i < 4; i--) // i = i - 1
   {
         cout << "Merhaba" << endl;
   }

   if (x == 1)
   {
       y = 5;
   }

   return 0;
}
Cevabi Goster

Hata: for dongusunde guncelleme kismi i-- olarak yazilmis. i degeri 1'den baslayip azaldigi icin i < 4 sarti her zaman dogru olur ve sonsuz donguye girer.

Cozum: i degeri artirilmali: i++

3. Soru (Atama vs. Karsilastirma)

#include <iostream>
#include <cmath>

using namespace std;

int main()
{
   int i;
   int x = 0;
   int y = 0;
   double z = 0.0;

   for (i = 1; i < 4; i++)
   {
         x = x + 1;
   }

   if (x = 1)
   {
       cout << "x'in degeri 1'dir!" << endl;
   }

   return 0;
}
Cevabi Goster

Hata: if (x = 1) kodu bir karsilastirma yapmaz, x'in degerini 1 yapar ve sarti hep dogru kabul eder.

Cozum: Esitlik kontrolu cift esittir ile yapilmali: if (x == 1)

4. Soru (Us Alma Parametreleri)

#include <iostream>
#include <cmath>

using namespace std;

int main()
{
   int i;
   int x = 2; 
   int y = 3; 
   double z = 0.0;

   for (i = 1; i < 4; i++)
   {
         z = pow(y, x); 
   }

   if (x == 2)
   {
       cout << "Sonuc: " << z << endl;
   }

   return 0;
}
Cevabi Goster

Hata: pow(y, x) kodu 3 uzeri 2 islemini (sonuc 9) hesaplar. Istenen islem ise 2 uzeri 3 idi.

Cozum: Degiskenlerin yeri degistirilmeli: z = pow(x, y);

Cevabi Goster

Hata: if (x > 5); satirinin sonundaki noktali virgul, if sartini o satirda bitirir. Altindaki parantezli kod blogu sarta bagli olmadan her turlu calisir.

Cozum: Noktali virgul kaldirilmali: if (x > 5) { ... }

6. Soru (Tam Sayi Bolmesi)

#include <iostream>
#include <cmath>

using namespace std;

int main()
{
   int i;
   int x = 5;
   int y = 2;
   double z = 0.0;

   for (i = 1; i < 4; i++)
   {
         y = y + i;
   }

   if (x == 5)
   {
       z = x / 2; 
       cout << "Bolme sonucu (2.5 bekleniyor): " << z << endl;
   }

   return 0;
}
Cevabi Goster

Hata: x ve 2 tam sayi oldugu icin z = x / 2; islemi ondalik kismi atarak 2 sonucunu verir.

Cozum: Sayilardan en az biri ondalikli olmali: z = x / 2.0;

7. Soru (Kapsam - Scope Kargasasi)

#include <iostream>
#include <cmath>

using namespace std;

int main()
{
   int i;
   int x = 1;
   int y = 0;
   double z = 0.0;

   for (i = 1; i < 4; i++)
   {
         int y = 5; 
         y = y + i;
   }

   if (x == 1)
   {
       cout << "y'nin son degeri: " << y << endl;
   }

   return 0;
}
Cevabi Goster

Hata: Dongu icinde int y = 5; diyerek gecici yeni bir degisken olusturulmus. Disaridaki asil y degiskeni bu islemden etkilenmez.

Cozum: Icerideki int kelimesi silinmeli: y = 5;

8. Soru (Mantiksal Operator - Aralik Kontrolu)

#include <iostream>
#include <cmath>

using namespace std;

int main()
{
   int i;
   int x = 25;
   int y = 0;
   double z = 0.0;

   for (i = 1; i < 4; i++)
   {
         y = i * 2;
   }

   if (x > 10 || x < 20)
   {
       cout << "x, 10 ile 20 arasindadir." << endl;
   }

   return 0;
}
Cevabi Goster

Hata: VEYA (||) kullanilmis. 25 degeri 10'dan buyuk oldugu icin ilk sarti saglar ve iceri girer.

Cozum: Iki sartin da saglanmasi icin VE kullanilmali: if (x > 10 && x < 20)

9. Soru (Negatif Sayinin Karekoku)

#include <iostream>
#include <cmath>

using namespace std;

int main()
{
   int i;
   int x = -16;
   int y = 0;
   double z = 0.0;

   for (i = 1; i < 4; i++)
   {
         y = y + 1;
   }

   if (x < 0)
   {
       z = sqrt(x); 
       cout << "Karekok: " << z << endl;
   }

   return 0;
}
Cevabi Goster

Hata: Negatif sayilarin karekoku gercek sayilarda alinamaz, program hata (NaN) uretir.

Cozum: Sayinin sifir veya pozitif oldugu kontrol edilmeli: if (x >= 0)

11. Soru (Hem Pozitif Hem Cift)

#include <iostream>
#include <cmath>

using namespace std;

int main()
{
   int i;
   int x = -4; 
   int y = 0;
   double z = 0.0;

   for (i = 1; i < 4; i++)
   {
         y = y + i;
   }

   if (x > 0 || x % 2 == 0)
   {
       cout << "Sayi hem pozitif hem de cifttir." << endl;
   }

   return 0;
}
Cevabi Goster

Hata: VEYA (||) kullanildigi icin, -4 gibi negatif ama cift bir sayi sarti saglar. Iki durumun da ayni anda gecerli olmasi gerekir.

Cozum: VE operatoru kullanilmali: if (x > 0 && x % 2 == 0)
