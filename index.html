C++ Hata Ayiklama Egzersizleri

Asagidaki kodlarin her birinde sadece bir tane mantik veya yazim hatasi bulunmaktadir. Kodun altindaki amaca ulasmasini engelleyen hatayi bulun ve "Cevabi Goster" butonuna tiklayarak kontrol edin.

1. Soru

Amac: 20 sayisinin karekokunu bulup ekrana yazdirmak.
#include <iostream>
#include <cmath>

using namespace std;

int main()
{
   int i;
   int x = 20;
   int y = 0;
   double z;

   for (i = 1; i < 4; i++)
   {
         y = x + i;
   }

   if (x == 20)
   {
       int z = sqrt(x); 
       cout << "Karekok: " << z << endl;
   }

   return 0;
}
Cevabi Goster

Hata: if blogunun icinde int z = sqrt(x); yazilmis. Hem z onceden double olarak tanimlanmisti, hem de sqrt ondalikli bir deger dondurur, int yapilmamalidir.

Cozum: Bastaki int kelimesi silinmeli: z = sqrt(x);

2. Soru

Amac: Ekrana tam 3 kere "Merhaba" yazdirmak.
#include <iostream>
#include <cmath>

using namespace std;

int main()
{
   int i;
   int x = 1;
   int y = 0;
   double z = 0.0;

   for (i = 1; i < 4; i--) // i = i - 1
   {
         cout << "Merhaba" << endl;
   }

   if (x == 1)
   {
       y = 5;
   }

   return 0;
}
Cevabi Goster

Hata: for dongusunde i-- kullanilmis. i 1'den baslayip azaldigi icin i < 4 sarti her zaman dogru olur ve sonsuz donguye girer.

Cozum: i artirilmali: i++

4. Soru

Amac: 2 uzeri 3 (2^3) islemini hesaplayip sonucu yazdirmak.
#include <iostream>
#include <cmath>

using namespace std;

int main()
{
   int i;
   int x = 2; 
   int y = 3; 
   double z = 0.0;

   for (i = 1; i < 4; i++)
   {
         z = pow(y, x); 
   }

   if (x == 2)
   {
       cout << "Sonuc: " << z << endl;
   }

   return 0;
}
Cevabi Goster

Hata: pow(y, x) 3 uzeri 2 hesaplar (sonuc 9). Bizden 2 uzeri 3 isteniyor.

Cozum: Parametrelerin yeri degistirilmeli: z = pow(x, y);

6. Soru

Amac: 5'i 2'ye bolerek tam ve ondalikli kismiyla (2.5) elde etmek.
#include <iostream>
#include <cmath>

using namespace std;

int main()
{
   int i;
   int x = 5;
   int y = 2;
   double z = 0.0;

   for (i = 1; i < 4; i++)
   {
         y = y + i;
   }

   if (x == 5)
   {
       z = x / 2; 
       cout << "Bolme sonucu: " << z << endl;
   }

   return 0;
}
Cevabi Goster

Hata: x ve 2 tam sayi oldugu icin islem ondalik kismi atarak 2 sonucunu verir (Tam sayi bolmesi).

Cozum: Sayilardan en az biri ondalikli belirtilmeli: z = x / 2.0;

   int x = -16;
   int y = 0;
   double z = 0.0;

   {

   }

   if (x < 0)
   {
       z = sqrt(x); 
       cout << "Karekok: " << z << endl;
   }

   return 0;
}
Cevabi Goster

Hata: if (x < 0) sarti yanlistir cunku negatif sayilarin karekoku programi hataya (NaN) surukler.

Cozum: Sayi sifirdan buyuk veya esit olmali: if (x >= 0)

10. Soru

Amac: Sayinin ayni anda hem pozitif hem de cift sayi oldugunu dogrulamak.
#include <iostream>
#include <cmath>

using namespace std;

int main()
{
   int i;
   int x = -4; 
   int y = 0;
   double z = 0.0;

   for (i = 1; i < 4; i++)
   {
         y = y + i;
   }

   if (x > 0 || x % 2 == 0)
   {
       cout << "Sayi hem pozitif hem de cifttir." << endl;
   }

   return 0;
}
Cevabi Goster

Hata: VEYA (||) kullanildigi icin -4 gibi negatif bir cift sayi sarti gecer.

Cozum: Iki durumun ayni anda gecerli olmasi icin VE kullanilmali: if (x > 0 && x % 2 == 0)

12. Soru

Amac: Girilen notun 1 ile 100 arasinda (1 ve 100 dahil) gecerli bir not olup olmadigini kontrol etmek.
#include <iostream>
#include <cmath>

using namespace std;

int main()
{
   int i;
   int x = 150; 
   int y = 0;
   double z = 0.0;

   for (i = 1; i < 4; i++)
   {
         y = y + i;
   }

   if (x >= 1 || x <= 100)
   {
       cout << "Gecerli bir not girdiniz." << endl;
   }

   return 0;
}
Cevabi Goster

Hata: VEYA (||) kullanildigi icin 150 degeri, 1'den buyuk olma sartini saglayip iceri girer.

Cozum: Belli bir aralik icin VE operatoru kullanilmali: if (x >= 1 && x <= 100)
