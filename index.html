C++ Hata Ayiklama Egzersizleri (Sadece IF ve Temeller)

Asagidaki kodlarin her birinde sadece bir tane mantik veya yazim hatasi bulunmaktadir. Kodun altindaki amaca ulasmasini engelleyen hatayi bulun ve "Cevabi Goster" butonuna tiklayarak kontrol edin.

1. Soru

Amac: Programin ekrana "Merhaba" yazdirip kapanmasi isteniyor.
#include <iostream>

using namespace std;

int main()
{
    cout << "Merhaba"

    return 0;
}
Cevabi Goster

Hata: cout satirinin sonunda noktali virgul ( ; ) eksik.

Cozum: Dogrusu cout << "Merhaba"; olmalidir.

2. Soru

Amac: Programin x degiskenine atanan 5 degerini ekrana yazdirmasi isteniyor.
#include <iostream>

int main()
{
    int x;

    x = 5;

    cout << x;

    return 0;
}
Cevabi Goster

Hata: Kutuphane eklendikten sonra using namespace std; komutu yazilmamis. Bu yuzden program cout komutunu taniyamaz.

3. Soru

Amac: Programin x degiskeni 10'a esit oldugunda ekrana "Sayi 10" yazdirmasi isteniyor.
#include <iostream>

using namespace std;

int main()
{
    int x = 10;

    if(x = 10)
    {
        cout << "Sayi 10" << endl;
    }

    return 0;
}
Cevabi Goster

Hata: if kosulunun icinde atama operatoru olan tek esittir ( = ) kullanilmis. Karsilastirma yapmak icin cift esittir ( == ) kullanilmalidir.

Cozum: Dogrusu if(x == 10) olmalidir.

4. Soru

Amac: Programin x ve y degiskenlerini toplayip sonucu ekrana yazdirmasi isteniyor.
#include <iostream>

using namespace std;

int main()
{
    int x;
    int toplam;

    x = 5;
    y = 3;

    toplam = x + y;

    cout << toplam;

    return 0;
}
Cevabi Goster

Hata: y degiskenine bir deger atanmis fakat kodun basinda tanimlanmamistir.

Cozum: Baslangica int y; eklenmelidir.

5. Soru

Amac: Sayi 10'dan buyukse ekrana "Buyuk", degilse "Kucuk veya Esit" yazdirsin.
#include <iostream>

using namespace std;

int main()
{
    int x = 5;

    if(x > 10)
    {
        cout << "Buyuk" << endl;
    }
    else(x <= 10)
    {
        cout << "Kucuk veya Esit" << endl;
    }

    return 0;
}
Cevabi Goster

Hata: else komutunun yanina sart parantezi ( ) yazilamaz. else, geri kalan tum durumlari kapsar.

Cozum: Ya sarti tamamen silip sadece else yazilmalidir, ya da else if(x <= 10) kullanilmalidir.

6. Soru

Amac: Programin isim degiskenine "Ali" kelimesini atayip ekrana yazdirmasi isteniyor.
#include <iostream>
#include <string>

using namespace std;

int main()
{
    string isim;

    isim = 'Ali';

Cevabi Goster

Hata: Metinler (string) cift tirnak ( " " ) icerisinde yazilmalidir. Tek tirnak ( ' ' ) sadece tek bir karakter (char) tanimlarken kullanilir.

Cozum: Dogrusu isim = "Ali"; olmalidir.

7. Soru

Amac: x degeri 0'dan buyukse ekrana alt alta "Pozitif" ve "Sayi" yazdirsin. Sifir veya negatifse HICBIR SEY yazdirmasin. (Su an x = -5)
#include <iostream>

using namespace std;

int main()
{
    int x = -5;

    if(x > 0)
        cout << "Pozitif" << endl;
        cout << "Sayi" << endl;

    return 0;
}
Cevabi Goster

Hata: if blogunda suslu parantez { } kullanilmadigi icin, if sarti sadece ilk satiri kapsar. Ikinci satir ("Sayi") if'ten bagimsizdir ve sayi negatif olsa bile her zaman ekrana yazdirilir.

Cozum: Iki cout islemini de suslu parantez { } icine almalisiniz.

8. Soru

Amac: Tanimlanan x degiskeninin degerini ekrana yazdirmak.
#include <iostream>

using namespace std;

int main()
{
    int x;

    cout << x;

    return 0;
}
Cevabi Goster

Hata: x degiskenine baslangic degeri atanmamistir. Bu durumda bilgisayar, bellekte o an bulunan rastgele (cop) bir degeri ekrana yazdirir.

Cozum: Tanimlarken veya hemen sonrasinda x = 0; gibi bir deger verilmelidir.

9. Soru

Amac: Programin x ve y degiskenlerini bolup, 2.5 olan kusuratli sonucu z'ye atayarak ekrana dogru yazdirmasi isteniyor.
#include <iostream>

using namespace std;

int main()
{
    int x = 5;
    int y = 2;
    double z;

    z = x / y;

    cout << z;

    return 0;
}
Cevabi Goster

Hata: x ve y tam sayi ( int ) olarak tanimlandigi icin bolme isleminin sonucu da tam sayi cikar (0.5 yutulur). z degiskeni double olsa bile islem coktan 2 olarak gerceklesmistir.

Cozum: Kusuratli sonuc almak icin islem sirasinda tur donusumu yapilmali: z = (double)x / y;

10. Soru

Amac: 20 sayisinin karekokunu bulup z degiskeni ile ekrana yazdirmak.
#include <iostream>
#include <cmath>

using namespace std;

int main()
{
   int x = 20;
   double z;

   if (x == 20)
   {
       int z = sqrt(x); 
       cout << "Karekok: " << z << endl;
   }

   return 0;
}
Cevabi Goster

Hata: if blogunun icinde int z = sqrt(x); yazilmis. Hem z onceden double olarak tanimlanmisti, hem de sqrt ondalikli bir deger dondurur, int yapilirsa kusurat kaybolur.

Cozum: Bastaki int kelimesi silinmeli: z = sqrt(x);

11. Soru

Amac: x sayisinin tam olarak 10 ile 20 arasinda (haric) oldugunu dogrulamak. (Su an x = 50)
#include <iostream>

using namespace std;

int main()
{
   int x = 50;

   if (10 < x < 20) 
   {
       cout << "Sayi 10 ile 20 arasindadir!" << endl;
   }

   return 0;
}
Cevabi Goster

Hata: Matematikteki gibi zincirleme kiyaslama C++'ta beklenen sonucu vermez. C++ bunu soldan saga isler: (10 < 50) kiyaslamasi dogrudur ve 1 sonucunu uretir. Sonra (1 < 20) kiyaslamasi da dogru cikar ve 50 sayisi sartlari saglamis gibi iceri girer.

Cozum: Araya VE operatoru konmalidir: if(x > 10 && x < 20)

12. Soru

Amac: x pozitifse "Pozitif", negatifse "Negatif", hicbiri degilse "Sifir" yazdirsin. (Su an x = 5)
#include <iostream>

using namespace std;

int main()
{
   int x = 5;

   if (x > 0)
       cout << "Pozitif" << endl;

   if (x < 0)
       cout << "Negatif" << endl;
   else
       cout << "Sifir" << endl;

   return 0;
}
Cevabi Goster

Hata: Ortadaki satira else if degil sadece if yazildigi icin, alttaki else sadece ikinci if'e baglanir. Bu durumda program x=5 oldugunda once "Pozitif" yazar, ardindan x 0'dan kucuk olmadigi icin else blogu calisir ve bir de "Sifir" yazar.

Cozum: Ikinci sart else if (x < 0) olarak degistirilmelidir.

13. Soru

Amac: 2 uzeri 3 (2^3) islemini hesaplayip sonucu yazdirmak. (8 bekleniyor)
#include <iostream>
#include <cmath>

using namespace std;

int main()
{
   int taban = 2; 
   int us = 3; 
   double z;

   z = pow(us, taban); 

   if (taban == 2)
   {
       cout << "Sonuc: " << z << endl;
   }

   return 0;
}
Cevabi Goster

Hata: pow fonksiyonuna parametreler ters verilmis. pow(us, taban) 3 uzeri 2 hesaplar ve sonuc 9 cikar.

Cozum: Parametrelerin yeri degistirilmeli: z = pow(taban, us);

Amac: Sadece x degeri 5'ten buyukse ekrana mesaj yazdirmak. (Su an x = 3)
#include <iostream>

int main()
{
   int x = 3;

   if (x > 5);
   {

Cozum: Noktali virgul silinmeli: if (x > 5) { ... }

Cozum: Buyuk veya esittir operatoru kullanilmali: if (yas >= 18)

#include <iostream>

using namespace std;

int main()
{
   int x = -16;
   double z;

   if (x < 0)
   {

}
Cevabi Goster

Hata: if (x < 0) sartiyla sadece negatif sayilarda islem yapilmaya calisiliyor, bu da C++ programini hataya (NaN) surukler.

19. Soru

Amac: Sayinin ayni anda hem pozitif hem de cift sayi oldugunu dogrulamak. (Su an x = -4)
#include <iostream>

using namespace std;

int main()
{
   int x = -4; 

   if (x > 0 || x % 2 == 0)
   {
       cout << "Sayi hem pozitif hem de cifttir." << endl;
   }

   return 0;
}
Cevabi Goster

Hata: VEYA (||) kullanildigi icin -4 gibi negatif bir cift sayi sadece cift olma kuralini karsilayarak sarti asar.

Cozum: Iki durumun ayni anda gecerli olmasi icin VE kullanilmali: if (x > 0 && x % 2 == 0)

20. Soru

Amac: x ile belirtilen gunun hafta sonu (Cumartesi yani 6 veya Pazar yani 7) olup olmadigini kontrol etmek.
#include <iostream>

using namespace std;

int main()
{
   int x = 6; 

   if (x == 6 && x == 7)
   {
       cout << "Bugun hafta sonu!" << endl;
   }

   return 0;
}
Cevabi Goster

Hata: VE (&&) kullanilmis. Degisken ayni anda hem 6 hem de 7 olamaz, bu sart asla gerceklesmez.

Cozum: Iki gunden biri olmasi yeterlidir: if (x == 6 || x == 7)

21. Soru

Amac: Girilen notun 1 ile 100 arasinda (1 ve 100 dahil) gecerli bir not olup olmadigini kontrol etmek. (Su an x = 150)
#include <iostream>

using namespace std;

int main()
{
   int x = 150; 

   if (x >= 1 || x <= 100)
   {
       cout << "Gecerli bir not girdiniz." << endl;
   }

   return 0;
}
Cevabi Goster

Hata: VEYA (||) kullanildigi icin 150 degeri, sirf 1'den buyuk olma sartini sagladigi icin iceri girer.

Cozum: Belli bir aralik icin VE operatoru kullanilmali: if (x >= 1 && x <= 100)
